# 文档架构规划

## 项目定位

Vxture Studio 总体规划是一个人工智能业务服务平台，提供智能体服务，包含以下核心系统：

1. **企业官网** - 企业宣传和业务展示门户
2. **运营服务平台** - 提供业务订阅、租户管理、账号管理、订单管理、工单管理等功能
3. **租户管理平台** - 提供租户管理、账号管理、业务订阅、工单提交、使用监测等功能
4. **业务应用平台** - 单独开发，不在本项目管理，提供具体业务功能
5. **公共接口** - 统一后端统一的接口服务
