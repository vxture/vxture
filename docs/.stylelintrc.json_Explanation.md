# .stylelintrc.json Configuration Explained

This document explains the structure and options of the `.stylelintrc.json` file in the Vxture project, focusing on modern Stylelint usage, Tailwind/Prettier integration, and best practices for CSS/SCSS linting.

## What is .stylelintrc.json?

`.stylelintrc.json` is the configuration file for Stylelint, a powerful linter for CSS and SCSS. It helps enforce consistent style, catch errors, and maintain best practices in your stylesheets.

## Example Configuration

```json
{
  "extends": [
    "stylelint-config-standard-scss",
    "stylelint-config-prettier-scss"
  ],
  "rules": {
    "selector-class-pattern": null,
    "scss/at-import-partial-extension": null,
    "scss/at-import-partial-extension-blacklist": null,
    "scss/dollar-variable-pattern": null,
    "media-feature-range-notation": null,
    "no-invalid-position-declaration": null,
    "scss/at-rule-no-unknown": [
      true,
      {
        "ignoreAtRules": [
          "tailwind",
          "apply",
          "variants",
          "responsive",
          "screen"
        ]
      }
    ]
  }
}
```

## Key Configuration Options

### Extends

- `stylelint-config-standard-scss`: Standard rules for SCSS syntax and best practices.
- `stylelint-config-prettier-scss`: Disables rules that conflict with Prettier formatting.

### Rules

- `selector-class-pattern`: `null` — No restriction on class naming patterns.
- `scss/at-import-partial-extension`: `null` — No restriction on SCSS partial import extensions.
- `scss/at-import-partial-extension-blacklist`: `null` — No blacklist for partial import extensions.
- `scss/dollar-variable-pattern`: `null` — No restriction on SCSS variable naming.
- `media-feature-range-notation`: `null` — No restriction on media feature range notation.
- `no-invalid-position-declaration`: `null` — No restriction on invalid position declarations.
- `scss/at-rule-no-unknown`: Allows custom at-rules for Tailwind CSS (`@tailwind`, `@apply`, `@variants`, `@responsive`, `@screen`).

## Tailwind CSS Integration

This config is tailored for Tailwind CSS. The `ignoreAtRules` option ensures that Tailwind-specific at-rules are not flagged as errors. Example:

```scss
.custom-button {
  @apply bg-blue-500 text-white py-2 px-4 rounded;

  &:hover {
    @apply bg-blue-700;
  }

  @screen md {
    @apply py-3 px-6;
  }
}
```

## Integration with Other Tools

### Prettier

Using `stylelint-config-prettier-scss` ensures Stylelint does not conflict with Prettier formatting.

### VS Code

1. Install the Stylelint extension.
2. Add to `.vscode/settings.json`:
   ```json
   {
     "stylelint.validate": ["css", "scss"],
     "editor.codeActionsOnSave": {
       "source.fixAll.stylelint": true
     }
   }
   ```

## Example: Valid SCSS

```scss
.feature-card {
  display: flex;
  margin-bottom: 2rem;

  &__icon {
    width: 48px;
    height: 48px;
    margin-right: 1rem;
  }

    flex: 1;

  &__title {

  }

    flex-direction: column;
    &__icon {
      margin-bottom: 1rem;
      margin-right: 0;
    }
  }

  @screen lg {
    margin-bottom: 3rem;
  }

```

## Best Practices

1. **Add Stylelint scripts**: In `package.json`:

   ```json
   {
     "scripts": {
       "lint:style": "stylelint 'src/**/*.scss' --fix"
     }
   }
   ```

2. **Use `.stylelintignore`**: Exclude files/folders from linting:

```text
node_modules/
build/
.next/
public/
```

3. **CI Integration**: Add style checks to your CI pipeline.

4. **Keep config up to date**: Review and update Stylelint config as the project evolves.

5. **Team consistency**: Ensure all team members use the same config for efficient code review.

## Troubleshooting

1. **Tailwind CSS errors**: Make sure all Tailwind at-rules are included in `ignoreAtRules`. Consider `stylelint-config-recommended-tailwindcss` for more advanced Tailwind support.
2. **Prettier conflicts**: Use `stylelint-config-prettier-scss` and run `npx stylelint-config-prettier-check` to check for conflicts.
3. **Performance**: Use `.stylelintignore` and enable caching (`--cache`, `--cache-location`) for large projects.
