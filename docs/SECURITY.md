# 安全与合规建议

## 身份与访问管理
- 使用 OAuth 2.0 / OIDC 做统一登录
- 使用 JWT 做服务间鉴权（签发短生命周期的 access token，refresh token 存储在安全存储）
- RBAC：基于角色的最小权限分配

## WAF 与网络边界
- 对外流量通过阿里云 WAF 进行防护
- 内部服务使用 Istio 的 mTLS 保证服务间通信加密

## 日志与审计
- 结构化日志(JSON)上报到 ELK / 阿里云 SLS
- 审计日志（用户敏感操作）需要单独保留与告警规则

## 密钥管理
- 使用阿里云 KMS / Jenkins 凭证插件管理密钥

## 合规性
- HTTPS 强制、证书到期自动更新（阿里云证书服务）
- 定期漏洞扫描与依赖项审计（Snyk / Dependabot / 内部扫描）
